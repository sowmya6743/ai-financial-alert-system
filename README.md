# 🤖 AI Financial Alert System

A Python-powered stock market monitoring system that fetches real-time stock prices and sends alerts when prices cross user-defined thresholds.

## 🚀 Features

- 📊 Fetches live stock prices from Yahoo Finance
- 🚨 Configurable price alerts for multiple stocks
- 📅 Timestamped scans (know exactly when data was fetched)
- 📦 Easy setup with `requirements.txt`

## 📈 Stocks Monitored

| Stock | Ticker | Alert Threshold |
|-------|--------|-----------------|
| Apple | AAPL | $280.00 |
| Google | GOOGL | $170.00 |
| Microsoft | MSFT | $430.00 |
| Amazon | AMZN | $190.00 |

## 🛠️ Tech Stack

- **Python 3.11** - Core programming language
- **yfinance** - Yahoo Finance API wrapper for stock data
- **Git** - Version control

## 📋 Prerequisites

- Python 3.11 or higher
- pip (Python package manager)
- Internet connection (for fetching live stock data)
